# 📚 Library Management System

A simple and interactive console-based Python application that allows users to manage their personal book collection and reading wishlist.

## 📌 Features
* **Book Inventory:** Add books you currently own to your library.
* **Wishlist Management:** Track books you want to read in the future.
* **Duplicate Prevention:** Prevents you from adding books to the wishlist that you already own.
* **Interactive Prompts:** Step-by-step user interface to handle data gracefully.

---

## 🚀 How to Run

### Prerequisites
* **Python 3.x** installed on your system.

### Steps
1. Clone or download the repository files.
2. Save the code in a file named `Library management.py`.
3. Run the script from the terminal:
   ```bash
   python "Library management.py"
